# Intents

Each file in this folder represents a single product intent — a capability the team has decided to build. Intents are written before elaboration begins and before any units are created.

**File naming:** `YYYY-MM-DD-[short-slug].md`

**Lifecycle:** Draft → Ready → In Elaboration → Elaborated → (units extracted to backlog)

Use `_template.md` to create a new intent.
